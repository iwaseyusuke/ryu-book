環境構築¶

本書では、Ryu、Open vSwitchおよびMininetの最新版がインストールされている環境を想定しています。

本書のための環境をもっとも簡単に整えるには、Dockerを利用することをお勧めします。

  • Ryu-Book 用のDocker imageを使う方法

    $ docker run -it --privileged -e DISPLAY=$DISPLAY \
             -v /tmp/.X11-unix:/tmp/.X11-unix \
             -v /lib/modules:/lib/modules \
             osrg/ryu-book

もし手動で環境を整えたい場合は、下記を参照してください。また、Open vSwitchおよびMininetのインストール時に問題が起きた場合は、それぞれのプロジェクトのHPを参照ください。

  • Ryu

    $ sudo apt-get install git python-dev python-setuptools python-pip
$ git clone https://github.com/osrg/ryu.git
$ cd ryu
$ sudo pip install .

  • Open vSwitch

    こちらのインストール手順参照ください。

  • Mininet

    こちらのインストール手順参照ください。
